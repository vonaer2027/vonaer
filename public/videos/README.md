# Video Assets

## Required Files

### hero-bg.mp4
- **Purpose**: Background video for hero sections across all minimal design variations
- **Used in**:
  - `/src/components/designs/minimal-1/hero.tsx`
  - `/src/components/designs/minimal-2/hero.tsx`
  - `/src/components/designs/minimal-3/hero.tsx`
- **Specifications**:
  - Format: MP4 (H.264 codec recommended)
  - Resolution: 1920x1080 or higher (4K recommended)
  - Duration: 10-30 seconds (will loop)
  - Aspect ratio: 16:9
  - File size: Under 10MB for optimal loading
  - Content: Luxury private jet footage, aerial views, or elegant travel scenes
  - Style: Dark/moody aesthetic to complement text overlays
