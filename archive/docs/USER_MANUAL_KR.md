## 1. 웹사이트 개요

### VONAER란?

VONAER는 아시아를 대표하는 프리미엄 항공 모빌리티 플랫폼입니다. 프라이빗 제트, 헬리콥터, Super Yacht, Chauffeured Car 서비스를 통해 고객에게 최상의 이동 경험을 제공합니다.

### 주요 서비스

- **Empty Leg**: 빈 항공편을 활용한 특가 프라이빗 제트 서비스
- **전용기 차터**: 원하는 일정과 노선으로 프라이빗 제트 이용
- **헬리콥터 서비스**: 도심 내 빠른 이동 및 관광 투어
- **Super Yacht**: 초호화 요트 전세 서비스
- **Chauffeured Car**: 프리미엄 차량 기사 서비스

## 2. 웹사이트 접속 및 기본 화면

### 웹사이트 접속

1. 웹 브라우저(크롬, 사파리, 엣지 등)를 실행합니다
2. 주소창에 VONAER 웹사이트 주소를 입력합니다
3. 엔터를 눌러 접속합니다

### 메인 화면 구성

웹사이트에 접속하면 다음과 같은 화면이 표시됩니다:

**상단 헤더 영역**
- 왼쪽: 메뉴 버튼 (세 개의 가로줄 모양)
- 중앙: VONAER 로고
- 오른쪽: 언어 선택, 테마 변경 버튼

**메인 콘텐츠 영역**
- VONAER 로고와 슬로건
- "Exploring The World With You" 문구
- "Book a Flight" 버튼 - 클릭하면 Empty Leg 페이지로 이동

**하단 영역**
- 연락처 정보
- 서비스 목록
- 회사 정보

## 3. 메뉴 사용법

### 메뉴 열기

1. 화면 왼쪽 상단의 메뉴 버튼(세 개의 가로줄)을 클릭합니다
2. 왼쪽에서 메뉴 패널이 슬라이드되어 나타납니다

### 메뉴 항목 설명

메뉴는 다음과 같은 항목으로 구성되어 있습니다:

| 메뉴 항목 | 설명 |
|----------|------|
| HOME | 메인 페이지로 이동 |
| CHARTER | 차터 서비스 하위 메뉴 (클릭하면 펼쳐짐) |
| EMPTY LEG | Empty Leg 항공편 목록 페이지 |
| PR | 보도자료 및 뉴스레터 |
| ABOUT US | 회사 소개 |
| CONTACT | 연락처 및 위치 정보 |
| MEMBER'S LOUNGE | 회원 전용 페이지 |

### CHARTER 하위 메뉴

CHARTER를 클릭하면 다음 항목이 펼쳐집니다:

- **JET & HELICOPTER**: 전용기 및 헬리콥터 차터
- **CHAUFFEURED CAR**: 프리미엄 차량 서비스
- **SUPER YACHT**: 초호화 요트 전세
- **eVTOL**: 미래 전기 수직 이착륙 항공기 정보

### 메뉴 닫기

- 메뉴 패널 오른쪽 상단의 X 버튼을 클릭합니다
- 또는 메뉴 패널 바깥 영역(어두운 배경)을 클릭합니다

## 4. Empty Leg 서비스 이용하기

Empty Leg는 VONAER의 핵심 서비스입니다. 전용기가 승객 없이 이동해야 하는 구간을 특별 할인가로 제공하는 서비스입니다.

### Empty Leg 페이지 접속 방법

**방법 1: 메인 화면에서 접속**
1. 메인 화면의 "Book a Flight" 버튼을 클릭합니다

**방법 2: 메뉴에서 접속**
1. 왼쪽 상단 메뉴 버튼을 클릭합니다
2. "EMPTY LEG"를 클릭합니다

### Empty Leg 페이지 화면 구성

페이지에 접속하면 다음 요소들을 확인할 수 있습니다:

**페이지 제목**
- "Empty Leg" 제목이 상단에 표시됩니다

**정렬 옵션**
- 항공편 목록을 원하는 기준으로 정렬할 수 있습니다
- 정렬 기준:
  - 가격 낮은순: 저렴한 항공편부터 표시
  - 가격 높은순: 비싼 항공편부터 표시
  - 날짜 가까운순: 출발일이 가까운 항공편부터 표시
  - 날짜 먼순: 출발일이 먼 항공편부터 표시

**항공편 카드 목록**
- 현재 이용 가능한 Empty Leg 항공편이 카드 형태로 표시됩니다

### 항공편 카드 정보 확인하기

각 항공편 카드에는 다음 정보가 포함되어 있습니다:

**항공기 이미지**
- 카드 상단에 해당 항공기 이미지가 표시됩니다

**항공기 기종**
- 예: Global 6000, Gulfstream G450 등

**노선 정보**
- 출발지: 출발 도시 및 국가
- 도착지: 도착 도시 및 국가
- 화살표로 이동 방향 표시

**비행 일자**
- 해당 Empty Leg 항공편의 출발 예정일

**탑승 인원**
- 최대 탑승 가능 인원

**가격 정보**
- 총 비행 비용 (미국 달러 기준)
- 일부 항공편은 "가격 문의"로 표시될 수 있습니다

**예약 문의 버튼**
- "예약 문의" 버튼을 클릭하여 해당 항공편에 대한 문의를 진행합니다

### 정렬 기능 사용하기

1. 페이지 상단의 정렬 선택 박스를 클릭합니다
2. 원하는 정렬 기준을 선택합니다:
   - 가격 낮은순
   - 가격 높은순
   - 날짜 가까운순
   - 날짜 먼순
3. 선택 즉시 항공편 목록이 해당 기준으로 재정렬됩니다

참고: 서울에서 출발하는 항공편이 우선적으로 상단에 표시됩니다.

## 5. 예약 문의 방법

원하는 항공편을 찾으셨다면, 예약 문의를 통해 상담을 받으실 수 있습니다.

### 예약 문의 절차

**1단계: 예약 문의 버튼 클릭**
- Empty Leg 페이지에서 원하는 항공편 카드의 "예약 문의" 버튼을 클릭합니다

**2단계: 예약 문의 창 확인**
- 팝업 창이 열리며 선택한 항공편 정보가 요약되어 표시됩니다
- 표시 정보: 항공기 기종, 출발지, 도착지, 비행 일자, 탑승 인원

**3단계: 연락처 정보 입력**

다음 정보를 입력해 주세요:

| 입력 항목 | 설명 | 필수 여부 |
|----------|------|----------|
| 이름 | 예약자 성함 | 필수 |
| 전화번호 | 연락 가능한 전화번호 | 필수 |
| 이메일 | 이메일 주소 | 선택 |

**4단계: 동의 항목 체크**

다음 두 가지 동의 항목에 체크해야 합니다:

1. **연락 동의**: "플라이트 예약에 관해 VONAER로부터 연락 받는 것에 동의합니다"
2. **개인정보 처리방침 동의**: 개인정보 처리방침 및 위치기반서비스 이용약관에 동의

각 약관은 파란색 링크를 클릭하여 내용을 확인할 수 있습니다.

**5단계: 요청 제출**
- 모든 정보 입력과 동의를 완료한 후 "리퀘스트 보내기" 버튼을 클릭합니다
- 버튼 클릭 후 잠시 기다려 주세요

**6단계: 제출 완료**
- "예약 문의가 성공적으로 전송되었습니다! 곧 연락드리겠습니다" 메시지가 표시됩니다
- VONAER 담당자가 입력하신 연락처로 연락을 드립니다

### 입력 시 주의사항

- 이름과 전화번호는 반드시 입력해야 합니다
- 전화번호는 정확하게 입력해 주세요 (연락 가능한 번호)
- 두 가지 동의 항목 모두 체크하지 않으면 제출이 불가합니다
- 제출 버튼 클릭 후 처리 중에는 창을 닫지 마세요

### 예약 문의 취소

- 예약 문의 창에서 "취소" 버튼을 클릭하면 창이 닫힙니다
- 입력한 내용은 저장되지 않습니다

## 6. 차터 서비스

VONAER는 다양한 종류의 항공기 차터 서비스를 제공합니다.

### 차터 서비스 페이지 접속

1. 메뉴를 열고 "CHARTER"를 클릭합니다
2. 펼쳐진 하위 메뉴에서 "JET & HELICOPTER"를 클릭합니다

### 항공기 종류

VONAER에서 제공하는 항공기 종류는 다음과 같습니다:

**라이트 제트 (Light Jet)**
- 탑승 인원: 4-8명
- 항속 거리: 1,500-2,500 nm
- 속도: 400-500 mph
- 특징: 단거리 여행에 비용 효율적, 소규모 공항 이용 가능

**미드 제트 (Mid Jet)**
- 탑승 인원: 6-10명
- 항속 거리: 2,000-3,500 nm
- 속도: 450-550 mph
- 특징: 넉넉한 공간과 편안함 제공

**헤비 제트 (Heavy Jet)** - 가장 인기 있는 옵션
- 탑승 인원: 8-16명
- 항속 거리: 3,500-7,000 nm
- 속도: 500-600 mph
- 특징: 대륙간 비행 가능, 풀 서비스 갤리 제공

**초장거리 제트 (Ultra Long Haul)**
- 탑승 인원: 10-19명
- 항속 거리: 6,000+ nm
- 속도: 550-650 mph
- 특징: 논스톱 대륙간 비행, 완전한 침실 제공

**VIP 에어라이너**
- 탑승 인원: 1-4명
- 항속 거리: 전 세계
- 특징: 궁극의 럭셔리 비행 경험

**헬리콥터**
- 탑승 인원: 2-8명
- 항속 거리: 200-400 nm
- 속도: 120-180 mph
- 특징: 도시 접근성, 헬리패드 이착륙 가능

### 항공기 정보 확인하기

각 항공기 카드에서 다음 정보를 확인할 수 있습니다:
- 항공기 이미지
- 기종 설명
- 탑승 인원, 항속 거리, 속도, 이용 가능 공항 수
- 주요 특징 목록

### 캐빈 레이아웃 보기

1. 원하는 항공기 카드에서 "View Cabin Layout" 버튼을 클릭합니다
2. 해당 항공기의 내부 좌석 배치도가 팝업으로 표시됩니다
3. 닫기 버튼을 클릭하여 팝업을 닫습니다

### 차터 문의하기

1. 원하는 항공기 카드에서 "문의하기" 버튼을 클릭합니다
2. 문의 양식이 팝업으로 표시됩니다
3. 필요한 정보를 입력합니다:
   - 이름 (필수)
   - 전화번호 (필수)
   - 이메일 (필수)
   - 문의 내용 (선택)
4. 동의 항목을 체크합니다
5. "문의하기" 버튼을 클릭하여 제출합니다

## 14. 자주 묻는 질문

### Q: Empty Leg란 무엇인가요?

A: Empty Leg는 전용기가 승객 없이 이동해야 하는 구간을 말합니다. 예를 들어, 서울에서 도쿄로 고객을 모신 후, 항공기가 다음 일정을 위해 다시 서울로 돌아와야 하는 경우, 도쿄에서 서울로 가는 빈 좌석을 특별 할인가로 제공하는 것입니다. 정규 차터 가격 대비 상당히 저렴하게 이용할 수 있습니다.

### Q: 예약 문의 후 얼마나 기다려야 하나요?

A: 예약 문의 제출 후 24시간 이내에 VONAER 담당자가 연락을 드립니다. 긴급한 문의는 전화 (+82 1600 9064)로 연락 주시기 바랍니다.

### Q: 가격은 어떻게 결정되나요?

A: Empty Leg 가격은 항공기 기종, 노선, 출발일 등에 따라 결정됩니다. 표시된 가격은 총 비행 비용이며, 일부 항공편은 "가격 문의"로 표시되어 직접 상담이 필요할 수 있습니다.

### Q: 탑승 인원을 초과할 수 있나요?

A: 각 항공기마다 최대 탑승 인원이 정해져 있습니다. 안전 규정상 이 인원을 초과할 수 없습니다. 인원이 많은 경우 더 큰 항공기를 이용하시거나 여러 대의 항공기를 예약해 주세요.

### Q: 예약 취소는 가능한가요?

A: 예약 취소 정책은 계약 조건에 따라 다릅니다. 자세한 내용은 예약 시 담당자에게 문의해 주세요.

### Q: 반려동물 탑승이 가능한가요?

A: 프라이빗 제트의 경우 반려동물 탑승이 가능한 경우가 많습니다. 다만, 항공기 및 노선에 따라 제한이 있을 수 있으니 예약 문의 시 담당자에게 확인해 주세요.

### Q: 결제 방법은 어떻게 되나요?

A: 결제 방법은 예약 확정 시 담당자를 통해 안내받으실 수 있습니다. 신용카드, 계좌이체 등 다양한 방법을 지원합니다.

### Q: 출발 전 무엇을 준비해야 하나요?

A: 여권 등 신분증을 지참해 주세요. 국제선의 경우 유효한 비자가 필요할 수 있습니다. 자세한 안내는 예약 확정 후 담당자가 안내해 드립니다.

### Q: 공항 라운지를 이용할 수 있나요?

A: VONAER는 VIP 라운지 서비스를 제공합니다. 자세한 내용은 예약 시 문의해 주세요.

### Q: 기내식이 제공되나요?

A: 항공기 기종 및 비행 시간에 따라 기내식이 제공됩니다. 특별 식단이 필요한 경우 예약 시 미리 요청해 주세요.
