## 1. 플랫폼 개요

### 1.1 VONAER란?

VONAER는 프라이빗 항공 서비스를 제공하는 온라인 플랫폼입니다. 고객은 웹사이트를 통해 Empty Leg 특가 항공편을 확인하고 예약할 수 있으며, 관리자는 항공편 정보를 관리하고 고객 예약 요청을 처리할 수 있습니다.

### 1.2 주요 서비스

- **Empty Leg 특가**: 빈 좌석을 활용한 프라이빗 항공편 특가
- **프라이빗 제트**: 맞춤형 프라이빗 제트 서비스
- **헬리콥터**: 헬리콥터 전세 서비스
- **eVTOL**: 전기 수직 이착륙기 서비스
- **요트/슈퍼요트**: 프라이빗 요트 전세
- **슈퍼카**: 럭셔리 차량 렌탈
- **프라이빗 출입국 서비스**: 전용 출입국 수속 지원

### 1.3 지원 언어

- 한국어
- 영어
- 일본어
- 중국어 (간체)

## 2. 고객용 기능

### 2.1 홈페이지

#### 2.1.1 메인 화면

**위치**: https://v1.vonaer.com/

**주요 요소**:
- 동영상 배경: 프라이빗 항공 서비스 소개 영상이 자동 재생됩니다
- VONAER 로고 및 서비스 소개
- "항공편 예약" 버튼: 클릭 시 예약 양식으로 이동

**사용 방법**:
1. 페이지 접속 시 자동으로 소개 영상이 재생됩니다
2. 화면 중앙의 "항공편 예약" 버튼을 클릭하면 예약 양식으로 스크롤됩니다
3. 상단 메뉴에서 다른 페이지로 이동할 수 있습니다

#### 2.1.2 언어 변경

**위치**: 화면 우측 상단

**사용 방법**:
1. 우측 상단의 언어 선택 버튼 클릭
2. 원하는 언어 선택 (한국어/English/日本語/中文)
3. 페이지가 자동으로 선택한 언어로 변경됩니다

#### 2.1.3 메뉴 탐색

**위치**: 화면 우측 상단 메뉴 버튼

**주요 메뉴**:
- Empty Leg: 특가 항공편 목록
- 항공기: 제트, 헬리콥터, eVTOL 서비스
- 요트: 요트 및 슈퍼요트 서비스
- 슈퍼카: 럭셔리 차량 렌탈
- 멤버십: 회원 프로그램 안내
- 회사 소개: VONAER 소개
- 연락처: 문의 정보

**사용 방법**:
1. 우측 상단 메뉴 버튼 클릭
2. 전체 화면 메뉴가 나타남
3. 원하는 메뉴 항목 클릭
4. 닫기 버튼(X)으로 메뉴 닫기

### 2.2 항공편 예약 양식

#### 2.2.1 예약 양식 작성

**위치**: 홈페이지 하단 또는 "항공편 예약" 버튼 클릭 후

**입력 항목**:

1. **여행 유형 선택**
   - 편도: 출발지에서 목적지까지 한 방향
   - 왕복: 출발지에서 목적지까지 갔다가 돌아오는 여정

2. **출발지**
   - 도시명 입력
   - 자동 완성 기능 지원
   - 예시: "서울", "인천", "김포"

3. **도착지**
   - 도시명 입력
   - 자동 완성 기능 지원
   - 예시: "도쿄", "오사카", "상하이"

4. **출발일**
   - 달력에서 날짜 선택
   - 오늘 이후 날짜만 선택 가능

5. **귀국일** (왕복 선택 시)
   - 달력에서 날짜 선택
   - 출발일 이후 날짜만 선택 가능

6. **탑승 인원**
   - 기본값: 1명
   - + 버튼: 인원 추가
   - - 버튼: 인원 감소
   - 최소: 1명

**사용 방법**:
1. 여행 유형 선택 (편도/왕복)
2. 출발지 입력 (도시명 입력 후 자동완성 목록에서 선택)
3. 도착지 입력
4. 출발일 선택 (달력 아이콘 클릭)
5. 왕복인 경우 귀국일 선택
6. 탑승 인원 조정 (+/- 버튼)
7. "검색" 버튼 클릭

#### 2.2.2 예약 요청 제출

**검색 버튼 클릭 후**:

1. **예약 요청 대화상자**가 나타남
2. **고객 정보 입력**:
   - 성함: 예약자 이름
   - 연락처: 휴대폰 번호 (예: 010-1234-5678)
   - 이메일: 이메일 주소 (선택사항)

3. **개인정보 수집 동의**:
   - 체크박스 선택 필수
   - 동의하지 않으면 예약 불가

4. **"예약 요청 제출" 버튼** 클릭

**예약 완료 후**:
- 성공 메시지 표시
- Google Chat으로 관리자에게 알림 전송
- 고객 정보는 데이터베이스에 저장
- 관리자가 연락처로 회신 예정

### 2.3 Empty Leg 특가 항공편

#### 2.3.1 Empty Leg이란?

Empty Leg는 프라이빗 제트가 고객을 목적지에 내려주고 돌아오거나, 다음 고객을 태우러 가는 빈 구간입니다. 이 구간을 특가로 제공하여 일반 전세 가격의 50-75% 저렴하게 이용할 수 있습니다.

#### 2.3.2 Empty Leg 항공편 보기

**위치**: https://v1.vonaer.com/empty

**주요 정보**:
- 항공편 사진
- 항공기 종류
- 출발지 / 도착지
- 출발 날짜
- 탑승 가능 인원
- 가격 (전체 기체 전세 요금)

**정렬 옵션**:
- 가격 낮은 순
- 가격 높은 순
- 날짜 빠른 순
- 날짜 늦은 순

**사용 방법**:
1. "Empty Leg" 메뉴 클릭 또는 URL 직접 접속
2. 항공편 목록 확인
3. 정렬 방식 선택 (기본: 날짜 빠른 순)
4. 관심 있는 항공편 카드 클릭
5. 상세 정보 확인

#### 2.3.3 항공편 예약하기

**항공편 카드에서**:

1. **"예약 요청" 버튼** 클릭
2. **예약 대화상자**에 고객 정보 입력:
   - 성함
   - 연락처
   - 이메일 (선택)
3. **개인정보 수집 동의** 체크
4. **"예약 요청 제출"** 클릭

**예약 후 프로세스**:
- 관리자가 Google Chat으로 알림 수신
- 담당자가 고객 연락처로 연락
- 항공편 이용 가능 여부 확인
- 최종 예약 확정 및 결제 안내

**참고사항**:
- Empty Leg는 선착순 마감
- 일정이 변경될 수 있음
- 정확한 출발 시간은 담당자 확인 후 결정

## 3. 관리자용 기능

### 3.1 관리자 로그인

#### 3.1.1 로그인 페이지 접속

**위치**: https://v1.vonaer.com/admin/login

**사용 방법**:
1. 웹브라우저에서 로그인 페이지 접속
2. 사용자명 입력
3. 비밀번호 입력
4. "로그인" 버튼 클릭

**보안 기능**:
- 세션 기반 인증
- 비밀번호 암호화
- 자동 로그아웃 (일정 시간 후)

#### 3.1.2 로그인 후 이동

로그인 성공 시 자동으로 관리자 대시보드로 이동합니다.

### 3.2 관리자 대시보드

#### 3.2.1 대시보드 개요

**위치**: https://v1.vonaer.com/admin

**주요 탭**:
1. 항공편 관리
2. 예약 요청
3. 사용자 관리
4. 가격 설정
5. MMS 발송

**공통 기능**:
- 새로고침: 최신 데이터 불러오기
- 로그아웃: 우측 상단 로그아웃 버튼

### 3.3 항공편 관리

#### 3.3.1 항공편 목록 보기

**위치**: 대시보드 > 항공편 관리 탭

**표시 정보**:
- 항공편 사진
- 항공기 종류
- 출발지 → 도착지
- 출발 날짜
- 가격 (원가 / 판매가)
- 탑승 인원
- 항공편 상태 (활성/비활성)

**정렬 및 필터**:
- 날짜순 정렬
- 활성/비활성 필터
- 출발지/도착지 검색

#### 3.3.2 새 항공편 추가

**사용 방법**:

1. **"+ 새 항공편" 버튼** 클릭
2. **항공편 정보 입력**:

   **기본 정보**:
   - 출발 도시: 출발 공항 또는 도시명
   - 출발 국가: 국가 선택
   - 도착 도시: 도착 공항 또는 도시명
   - 도착 국가: 국가 선택
   - 출발 날짜: 달력에서 선택
   - 항공기 종류: 기종명 (예: Gulfstream G650)

   **가격 정보**:
   - 기본 가격: 원가 (USD)
   - 가격 유형:
     - 고정 가격: 확정된 가격
     - 문의: "Enquire for Price" 표시
   - 맞춤 가격: 판매가 설정 (선택사항)

   **탑승 정보**:
   - 탑승 인원: 최대 탑승 가능 인원수

   **이미지**:
   - 항공편 사진 업로드
   - 여러 장 업로드 가능
   - 권장 크기: 최소 1200x800px

3. **"저장" 버튼** 클릭

**자동 처리**:
- 항공편 ID 자동 생성
- 생성 시간 자동 기록
- 기본 상태: 활성

#### 3.3.3 항공편 수정

**사용 방법**:

1. 수정할 항공편 카드에서 **"수정" 버튼** 클릭
2. 항공편 정보 수정
3. **"저장" 버튼** 클릭

**수정 가능 항목**:
- 모든 항공편 정보
- 이미지 추가/삭제
- 가격 변경
- 상태 변경 (활성/비활성)

**참고사항**:
- 수정 시 최종 수정 시간 자동 업데이트
- 고객 웹사이트에 즉시 반영

#### 3.3.4 항공편 삭제

**사용 방법**:

1. 삭제할 항공편 카드에서 **"삭제" 버튼** 클릭
2. 삭제 확인 대화상자에서 **"확인"** 클릭

**삭제 방식**:
- 소프트 삭제: 데이터베이스에서 비활성화
- 활성 목록에서 제거
- 고객 웹사이트에서 숨김

**복구 방법**:
- 데이터베이스에서 직접 복구 가능
- 관리자에게 문의

#### 3.3.5 가격 마진 설정

**위치**: 대시보드 > 가격 설정 탭

**기능**: 모든 항공편에 일괄 마진 적용

**사용 방법**:

1. **마진 퍼센트 입력**
   - 예시: 10% 입력 시 원가의 110% 판매
   - 20% 입력 시 원가의 120% 판매

2. **"마진 적용" 버튼** 클릭

3. **적용 결과 확인**:
   - 모든 항공편의 판매가 자동 계산
   - 고객 웹사이트에 마진 적용된 가격 표시

**예시**:
- 원가: $10,000
- 마진: 15%
- 판매가: $11,500

**참고사항**:
- 개별 항공편의 "맞춤 가격" 우선 적용
- 맞춤 가격이 없는 경우만 마진 적용

### 3.4 예약 요청 관리

#### 3.4.1 예약 요청 목록

**위치**: 대시보드 > 예약 요청 탭

**표시 정보**:
- 고객명
- 연락처

   - 귀국편 비행 시간
   - 귀국편 항공기 정보

4. **수신자 선택**
   - 고객 목록에서 체크박스 선택


1. FlyXO 및 JetBay 사이트 접속

- 부적절한 항공편 삭제

## 5. 자주 묻는 질문

### 5.1 고객용 FAQ

**Q1: Empty Leg 항공편은 언제 업데이트되나요?**

A: Empty Leg 항공편은 실시간으로 업데이트됩니다. 새로운 항공편이 추가되면 즉시 웹사이트에 표시되며, 자동 크롤링 시스템이 매일 여러 번 실행되어 최신 정보를 제공합니다.

**Q2: 예약 요청 후 얼마나 기다려야 하나요?**

A: 일반적으로 영업시간 내에 30분~2시간 이내에 연락드립니다. 업무 시간 외 요청은 다음 영업일에 연락드립니다.

**Q3: Empty Leg 가격은 최종 가격인가요?**

A: 웹사이트에 표시된 가격은 기체 전체 전세 요금입니다. 추가 비용(공항 수수료, 케이터링 등)이 있을 수 있으므로 정확한 최종 금액은 담당자 확인 후 안내됩니다.

**Q4: 예약을 취소할 수 있나요?**

A: 예약 취소 정책은 항공편 및 예약 조건에 따라 다릅니다. 담당자와 예약 확정 시 취소 정책을 확인하시기 바랍니다.

**Q5: 언어를 변경하면 다시 로그인해야 하나요?**

A: 아니요. 언어 변경 시 페이지가 자동으로 새로고침되며 세션은 유지됩니다.

### 5.2 관리자용 FAQ

**Q1: 항공편을 삭제하면 복구할 수 있나요?**

A: 삭제된 항공편은 비활성화 상태로 데이터베이스에 보관됩니다. 데이터베이스에서 직접 복구하거나 시스템 관리자에게 문의하세요.

**Q2: 가격 마진을 변경하면 기존 항공편에도 적용되나요?**

A: 네, 마진을 변경하면 모든 활성 항공편에 즉시 적용됩니다. 단, 개별 항공편에 "맞춤 가격"이 설정된 경우 해당 항공편은 맞춤 가격이 우선 적용됩니다.

**Q3: MMS 발송 비용은 어떻게 되나요?**

A: MMS 발송 비용은 BIZGO 플랫폼의 요금제에 따릅니다. 이미지가 포함된 MMS는 건당 약 30-50원입니다. 정확한 비용은 BIZGO 관리자 페이지에서 확인하세요.

**Q4: 자동 크롤링이 중복된 항공편을 추가하나요?**

A: 아니요. 크롤링 시스템은 자동으로 중복을 감지하고 제거합니다. 동일한 노선, 날짜, 가격의 항공편은 한 번만 추가되며, 최신 정보가 우선됩니다.

**Q5: 예약 요청이 Google Chat으로 알림이 안 오면 어떻게 하나요?**

A: Google Chat Webhook URL이 올바르게 설정되었는지 확인하세요. 환경 변수 설정을 확인하거나 시스템 관리자에게 문의하세요.

**Q6: 과거 날짜의 항공편을 자동으로 삭제할 수 있나요?**

A: 현재는 수동으로 비활성화해야 합니다. 하지만 고객 웹사이트에서는 자동으로 과거 날짜 항공편이 숨겨지므로 고객에게는 표시되지 않습니다.

**Q7: 관리자 계정을 추가할 수 있나요?**

A: 현재 버전은 단일 관리자 계정만 지원합니다. 추가 계정이 필요한 경우 시스템 관리자에게 문의하여 데이터베이스에서 직접 추가해야 합니다.

## 부록 A: 용어 사전

- **Empty Leg**: 프라이빗 제트가 승객 없이 이동하는 구간으로, 특가로 제공되는 항공편
- **전세**: 항공기 전체를 빌리는 것 (좌석 단위가 아님)
- **원가**: 항공사 또는 브로커가 제시한 기본 가격
- **판매가**: 마진이 포함된 최종 고객 가격
- **마진**: 원가에 추가되는 수수료 비율
- **맞춤 가격**: 개별 항공편에 설정한 특별 가격
- **활성/비활성**: 항공편의 표시 여부 (활성: 고객에게 보임, 비활성: 숨김)
- **크롤링**: 외부 웹사이트에서 자동으로 정보를 수집하는 과정
- **MMS**: 멀티미디어 메시지 (이미지 포함 문자메시지)
- **SMS**: 단순 문자메시지 (이미지 없음)
